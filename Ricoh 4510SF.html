Painel

Como configurar a rede da impressora:

  1. Pressione o botão User Tools
  2. Definições do sistema;
  3. Definições de interface;
  4. Rede;
  5. Endereço IPV4 da máquina;
  6. Verifique se especionar está selecionado
  7. Clique em edereço IPV4 da máquina, altere e clique em OK
  8. Clique em máscara de sub-rede, altere e clique em OK
  9. Clique em OK
  10. Clique na opção endereço do gateway IPv4
  11. Altere o endereço do gateway IPv4 e clique em OK
  12. Pressione sair ate voltar para o menu inicial


Modo técnico da impressora:

  1. Aperte o botão Reset (botão amarelo);
  2. Digite 8 0 6 1 8 2;
  3. Segure o botão C (Clear), até a tela mudar.

Reset erro de fusão (SC 541 até o 546)

  1. Clique em System Sp;
  2. Digite 7 8 0 7 # ;
  3. Clique em Execute e depois Exit;
  4. Reinicie o equipamento.

Reset PCDU - Unidade do Cilindro ou Tambor

  1. Clique em System Sp;
  2. Digite 7 8 0 4 0 1 1 # ;
  3. Clique em Execute e depois Exit;
  4. Reinicie o equipamento.


Erro SC332-00

Este erro aparece quando o toner não está chegando até a unidade de imagem.

Possiveis causas:

  1. Não foi retirado o lacre do cartucho após a troca;
  2. Cartucho com defeito;
  3. Unidade de imagem com defeito;
  4. Solenoide com defeito;
